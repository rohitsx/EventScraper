## Setup Instructions  

### Server Setup  

```bash
git clone https://github.com/devrohit/EventScraper.git
cd EventScraper/server
docker-compose up -d
```


### clinet setup 

```bash
cd EventScraper/client
pnpm i
dev: pnpm run dev
build: pnpm run build
```
